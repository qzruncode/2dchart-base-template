## ewchart-base-template

1. 此项目为ewchart脚手架的基础模板
2. 使用[dark-tunnel](https://www.npmjs.com/package/dark-tunnel)下载此模板，运行如下命令

  ```
  mkdir ewchartDemo
  cd ewchartDemo
  dark init ewchart-base-template
  npm install
  npm start
  ```
